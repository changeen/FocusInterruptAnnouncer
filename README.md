# FocusInterruptAnnouncer
## 🤖 开发说明
由于目前得知的消息，12.0版本后wa插件不再更新，

依赖于wa插件的焦点播报相关插件可能无法使用，遂借助AI编写此插件。

**本插件使用 DeepSeek AI 辅助编写**


**特别欢迎贡献者完善：**
- 🌐 多语言支持（目前只有中文）
- ⚡ 性能优化
- 🐛 Bug 修复
- ✨ 新功能建议和帮助开发，例如鼠标移至目标姓名半自动标记等。

## 功能特性
- 设置焦点后在/说频道播报。

## 安装方法
1. 下载 Releases 中的压缩包
2. 解压到 `World of Warcraft\_retail_\Interface\AddOns\`
3. 重新加载界面（/reload）
4. 输入/fia，打开配置面板，设置冷却时间，自定义喊话内容
