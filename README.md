# FocusInterruptAnnouncer
## 🤖 开发说明
**本插件使用 DeepSeek AI 辅助编写**，已实现基本功能，但仍有优化空间。

**特别欢迎贡献者完善：**
- 🌐 多语言支持（目前只有中文）
- ⚡ 性能优化
- 🐛 Bug 修复
- ✨ 新功能建议

## 功能特性
- wow焦点打断插件，设置焦点后在/说频道播报。

## 安装方法
1. 下载 Releases 中的压缩包
2. 解压到 `World of Warcraft\_retail_\Interface\AddOns\`
3. 重新加载界面（/reload）
4. 输入/fia，打开配置面板，设置冷却时间，自定义喊话内容
